# **App Name**: Lucky THR Multi-User

## Core Features:

- Event Landing Page: User-facing landing page explaining the Lucky THR concept with a clear call-to-action for administrators to create their event.
- Admin Panel & Event Setup: Secure administrator login (including Google Login) to create new events, customize titles, personalized messages, configure available THR nominals, and obtain a unique sharing link.
- Real-time Winner Monitoring: Live updating dashboard view for administrators to track participants, view their submitted details, awarded THR amounts, and easily copy account numbers for swift transfers.
- User Participation Form: Dynamic user page accessible via event link where participants enter their name, upload a facial photo, and optionally provide bank/e-wallet details before playing.
- Interactive Spin Wheel: An engaging, animated spin wheel experience that randomly determines the participant's THR amount based on the event's configured nominal values.
- Personalized Result Card: Displays the participant's photo, the awarded THR amount, a custom message, and provides options to save the result as an image or share it via WhatsApp with a pre-filled template.
- Anti-Cheat & Fairness System: Backend mechanism to prevent duplicate participation by checking against historical user/IP records, ensuring each participant receives only one THR per event.

## Style Guidelines:

- Light color scheme, conveying warmth and festivity while maintaining a clean aesthetic for mobile-first interaction.
- Primary color: A vibrant golden-orange (#E6C24C) representing luck, prosperity, and celebration, providing a warm, inviting feel.
- Background color: A subtly desaturated cream (#F9F7F2) derived from the primary hue, ensuring readability and a soft, modern base.
- Accent color: A rich, slightly bolder orange (#E1570E) for prominent call-to-action elements, offering a strong contrast and visual hierarchy.
- Font: 'Inter' (sans-serif) for all text. Its modern, clean, and highly readable characteristics provide an objective and contemporary feel suitable for both headlines and body text on mobile devices.
- Use clear, intuitive icons consistent with modern native app design principles, focusing on functionality like gifts, sharing, security, and profile actions to enhance user experience.
- Mobile-first responsive design, employing card-based structures and generous spacing for an uncluttered interface. Key information and interactive elements are strategically placed for easy access on small screens.
- Implement smooth and delightful animations, particularly for the spin wheel (8-10 seconds duration), page transitions, and interactive elements to provide a 'native app' feel and engaging user feedback.
